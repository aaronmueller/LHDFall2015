The IRC channel is a great place to talk to other developers that use Panda3D, and to ask questions about anything related to Panda3D.

Network FreeNode
Server chat.freenode.net
Channel #panda3d

You need an IRC client to connect to the channel. If you do not have an IRC client, you can use the web client: http://webchat.freenode.net/?channels=panda3d

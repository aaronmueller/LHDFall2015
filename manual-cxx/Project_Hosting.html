Panda3D Manual: Project Hosting

   Prev Top Next   

If you want some webspace to host your Panda3D project, you can do so at Panda3DProjects. Panda3DProjects is a website set up by the community and for the community. It provides free hosting for any Panda3D user who wants some room to put their Panda3D-related stuff. It also provides a wiki for each user where the user can create pages and show their progress.

You can visit Panda3DProjects at one of these links:

  • www.p3dp.com
  • www.panda3dprojects.com

An IRC channel exists for Panda3DProjects; #panda3dprojects at irc.freenode.net.

Finally, there is a TeamSpeak server available. The server name is p3dp.com and the port settings are the default.

Open source projects can also be hosted for free at SourceForge, Google Code or GitHub, among other places.
